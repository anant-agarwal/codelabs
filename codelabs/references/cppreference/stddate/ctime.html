ctime
Syntax: 
  #include <ctime>
  char *ctime( const time_t *time );

The ctime() function converts the calendar time time to local time of the format:

   day month date hours:minutes:seconds year

using ctime() is equivalent to

   asctime( localtime( tp ) );
